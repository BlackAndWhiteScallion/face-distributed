required dependencies:

opencv
numpy
dlib
imutils
psycpog2
flask
glob
uuid

to run the website, first navigate to the database folder, then run
```bash
python create.py
```

then, to run the website, run
```bash
python app.py
```

if the database is too populated, navigate to the database folder, run
```bash
python delete.py
```

the individual frames and the processed video can be found in the uploads folder
